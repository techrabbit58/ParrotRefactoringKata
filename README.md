# Parrot Refactoring Kata

__Task:__ Refactor the original code, take small steps, run the tests often. 
See how small and beautiful and polymorphic you can make it.

There is a Guided Learning Hour using this exercise: 
[Everyday Design Improvements in OO Code](https://youtu.be/NCEWAqEFPic)

## Acknowlegements

This code is heavily inspired by one of the examples in Martin Fowler's book "Refactoring".
