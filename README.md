# Python version of the Parrot Refactoring Kata

Ref.: [Emily Bache's Parrot Refactoring Kata](https://github.com/emilybache/Parrot-Refactoring-Kata/tree/main/Python)

It is recommended to create a virtual python environment for this project.
You will need at least version 3.10 of Python. Then install the dependencies:

    uv add pytest

Run the tests:

    uv run pytest src/parrot
